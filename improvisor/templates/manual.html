Authentication & User Settings

Registration

To be able to use Improvisor, you need to sign up for an account. This will enable you to download your own assets that will not be available to other users. The registration process is straightforward and only require the full name, email address and a password.

Login

To log into the account the email address and password that were used in the registration process will be required.

Password Reset

If the you forget your password, it can be easily reset by clicking on the 'Forgot Your Password?' link on the Login page. Email address, under which the profile was registered, will need to be provided, to which a password reset link will be sent. By clicking on the link, you will be asked to provide a new password.

User Settings

Once logged in, the user settings can be accessed by clicking either on the user name in the navigation bar or on the user photo. Here email address, name and password can be changed, as well as the colour of the presenter background.

Assets

Assets are any png, jpeg, text files and links, that you want to use in your presentation. Added assets are stored on the server and are ready to be used in any presentation. You can access the assets page by clicking on the 'Asset Pool' button on the right-hand side of the main page.

Adding New Assets

To add a new asset go to the 'Asset Pool' page and click the 'Add' button that is situated in the right top corner. On the next page enter the asset name and asset tags that will be associated with it. Next, you can choose if you want the asset to be a file (pdf, jpeg, png) or a link. As a default, it is set to file. Then choose a file, or enter http(s) address of the link. Once the file has been selected, or link entered, a default thumbnail will be displayed. If you want the name of the asset to be used as a thumbnail tick the 'Use name as thumbnail' checkbox. Alternatively, you can click on the displayed image and select a different file that will be used as a thumbnail.

If you would like to add multiple assets at once, you can click on the '+ Add Asset' button which will display an additional form for the next asset. Once you have completed all fields for all assets, you can save them by clicking on the 'Submit' button.

To make sure that the selected tags will be recognised by speech recognition, you can test them by clicking on the 'Test Recognition' button in the bottom right corner of the screen. Then say a sentence containing your tag and see if the output contains the tag that you want to use.

Editing Assets

Added assets can be deleted, or edited. To access this feature, go to 'Asset Pool' where all saved assets will be displayed. The assets can be sorted by when they were added ('Most Recent' or 'Oldest First'), or by relevance. To sort by relevance tags need to be provided in the 'Filter Tags:' input box. When more than one tag is provided, only assets that are linked to all tags will be displayed.

To delete an assets you can click on it, which will display its details, and then click on 'Delete Asset'. Alternatively, click on the 'Select' button, that is situated above the box that displays all assets, and then select an asset to delete. A 'Delete' button will appear next to the 'Select' button. Clicking on the 'Delete' button will delete the selected asset. By using the 'Select' button you can highlight multiple assets and delete all of them at the same time.

Clicking on a particular asset will open a page with the asset details, where changes can be applied. Here you can modify the thumbnail, by clicking on it and selecting a new thumbnail file, or ticking the 'Use name as thumbnail' checkbox. Additionally, the name can be updated (by clicking on it) and tags can be added or deleted. When adding new tags, you can test if they will be recognised properly, using the 'Test Recognition' in the bottom right corner of the screen.

Sessions

Active Session

A session represents one presentation. Once a session has been started it can be paused and then restarted at any time. The session will remain active until a new session is started. When a new profile is created, only a new session can be started by clicking on the 'New Session' button in the main menu. If there are already previous sessions, an option to 'Continue Session' will become available. A particular session will remain active until a new one is started.

Session Modes

When a new session is started, or an option to continue the previous session is selected, you will be given two mode choices: 'Presenter' and 'Controler'.

The 'Presenter' mode is the area where assets (slides) will be displayed. This mode should be open on the main display, that is dedicated to the audience.

The 'Controler' mode should be open on a mobile device, that will be used to control what is displayed for the audience. This is the main view that will be used during a presentation. When the mode is entered the speech recognition is, by default, switched off. To switch this feature on click the microphone icon in the top right corner. Clicking the icon again will switch the recognition off.

The main part of the screen in the 'Controler' mode is divided into 3 parts. The first part is the 'Current' list. All recognised tags will be displayed there, ordered by the number of times each tag has been mentioned. The assets whose tags were mentioned the most will be located on the left side of the list. To select an asset from the list click on a thumbnail, which will bring it to the foreground and will make it bigger, then swipe the thumbnail up to display it on the presenter. Selecting an asset from the 'Current' list will clear the list (the list will not be cleared until the next tag is recognised).

The second part of the 'Controler' view is dedicated to a 'Frequent' list. As with the previous list, all assets linked to the mentioned tags will be added here and ordered by the number of mentions. The assets can be selected from the list in the same way as from the 'Current' list. The main difference with this list is that selecting an asset will not clear it and new items will continue to be added.

The last part of this mode view is a tab section. By default, there is only one tab and you can add another 4 tabs by clicking on the plus button. The currently selected tab can be recognised by a white background button with blue text. The inactive tabs have the colours reversed. Each displayed asset will appear in the active tab so it can be easily seen which assets have already been shown. They can also be used to re-displayed a previously shown asset, or simply to break up the used assets into separate tabs. The assets are selected from the tabs in the same way as from the two previous lists.

In a case where the speech recognition is unable to recognize a tag, you can look for it manually. To do this click on the magnifying glass icon in the middle of the navigation bar. Pressing the button will open a modal with an input box and a button. First, enter the tag and then press the 'Go' button. Assets associated with the tag will be displayed in a list. Click a thumbnail of an asset you want to show to the audience and once it is highlighted swipe it up.

Previous Sessions

All previous sessions are saved and can be accessed from the main menu by clicking on 'Previous Sessions'. You will be presented with a list of all sessions which can be sorted by their date. The last session on the list (the most recent one) will always be active, and the status will change to finished only when a new one is started.

Clicking on any of the sessions from the list displays the session's details. Here you can amend the session's name (by clicking on it), export assets shown during the session in a pdf file, or delete the session.
